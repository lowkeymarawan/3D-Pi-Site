# 3D-Pi Front-End

Premium React landing page/front-end concept for 3D-Pi, a B2B SaaS platform for 3D printing businesses.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Design direction: enterprise consulting/SaaS visual language inspired by Bain, DON Global, BRG, and Booz Allen — dark executive hero, strategic messaging, modular platform cards, dashboard preview, insights section, and strong demo CTA.
